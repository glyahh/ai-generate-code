# new-techno

用法：当用户说“最新技术/前沿/有什么新东西/趋势”时使用。

工作流：
1) 明确主题（例如：React/Java/Spring/LLM/RAG/Agent）。
2) 优先用 MCP `context7` 拉取最新权威资料/版本信息。
3) 产出：
   - 近 3-5 个值得关注点
   - 适用场景与落地建议
   - 你当前项目的迁移建议（如有仓库上下文）
