# humanizer_codex

用法：当用户说“润色/口语化/更像人写的/去 AI 味/改写”，或直接输入“/humanizer_codex”时使用。

工作流：
1) 先问清楚：目标读者、语气（正式/半正式/口语）、语言（中/英）。
2) 保留事实不变，只改表达；避免夸张营销腔。
3) 输出：
   - 改写后的正文
   - （可选）3-5 条可替换表达

注意：
- 不添加用户未提供的事实或数据。
